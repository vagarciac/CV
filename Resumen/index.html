Desarrollos Recientes

Contabilidad Electrónica para el SAT

Es un aplicativo para resolver el problema de validar previamente la contabilidad electrónica enviada al SAT.

El aplicativo recibe la contabilidad electrónica en archivos Excel. La salida son archivos XML validados contra las especificaciones proporcionadas por el SAT.

Internamente se convierten los Excel en XML. Los XML se validan contra las especificaciones del SAT y se formatean para desplegarlos de forma adecuada. Adicionalmente se permite al usuario bajar el XML a su equipo local en formato zip como lo solicita el SAT y en paralelo se mantiene un histórico de los archivos XML generados.

La primera versión la desarrollé con tecnología ASP.Net. La versión final la desarrollé con ASP.Net MVC. Adicionalmente publiqué el aplicativo en una máquina virtual de Azure. Utilicé TFS para el control de versiones. La versión final es responsiva con el objetivo que pueda ser desplegada en cualquier dispositivo móvil o de escritorio.
