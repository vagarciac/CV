Recent Developments

Electronic Accounting for the SAT

Is an application to solve the problem of the digital accounting validation previously sent to the SAT.

The application receives the digital accounting in Excel files. The output XML files are validated against the specifications provided by the SAT.

The application turn Excel files into XML files. XML files are validated against SAT specifications and formatted appropriately for the final user. Additionally the user is allowed to download the XML file to your local computer in zip format as requested by the SAT and simultaneously a XML files historic is generated.

The first version was developed with .NET technology. Additionally, the application was published in a Azure virtual machine. I used TFS as a software control versions. The final version is responsive to ensure the application can be deployed on any mobile device or desktop.
